The Beatles - "All You Need Is Love"

The Song Lyrics

This simple chart shows all the words and how many times they are repeated throughout the lyrics. I made this for a Data Structures class at Brooklyn College

This chart was produced in the following manner:

  • The code for finding the frequency of each word is a simple implementation of a HashMap in Java
  • The HashMap is then output to a .csv file
  • The .csv file is processed by another Java class which utilizes Tablesaw dataframe and vizualization library, which in turn uses Plot.ly to produce graphs like this one.
